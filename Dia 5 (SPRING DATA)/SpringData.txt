    ANOTACIONES
        @Query
            Para poner la query de nuestra funcion, en caso de que sea SQL tendremos que poner nativeQuery=true
        @Param
            En caso de que en @Query tengamos para parametros
            Ejemplo
                @Query(select u from User where u.firstname = :firstname)
                User findByFirstName(@Param("firstname") String firstname)
